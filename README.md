# SKNETZERO

## Getting started

```
npm install
npm start
```

## 디렉토리 구조

- 디렉토리 기본 구성은 아래와 같으며, 아토믹 디자인 기반으로 구성되어 있습니다

```
- src/
--- assets/
--- components/
----- atoms/
----- molecules/
----- organisms/
----- pages/
--- hocs/
--- routers/
--- theme/
--- utils/
```

## 기본 구조

- component.tsx : 비즈니스 로직
- components.presenter.tsx : style 로직

## 페이지 이동

- routers 폴더 내부에 router관련 코드 포함되어 있습니다

## 그 외

- theme 폴더와 components > atoms 내부에 컴포넌트 관련 작업 되어있습니다
